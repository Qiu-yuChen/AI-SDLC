# 图书管理系统 产品规格说明书（草稿）

## 1. 概述
本系统旨在提供一个基础的图书管理功能，支持图书的增删改查操作。

## 2. 功能需求
- 图书信息管理：支持添加、编辑、删除图书信息。
- 图书检索：支持按书名、作者、ISBN等字段搜索图书。
- 图书状态管理：可标记图书为“可借”、“已借出”等状态。

## 3. 数据实体（初步）
- 图书：书名、作者、ISBN、出版社、出版日期、状态（可借/已借出）。
