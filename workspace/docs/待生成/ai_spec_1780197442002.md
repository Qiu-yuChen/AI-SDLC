# 员工车辆预约系统 规格说明书（草稿 v0.1）

## 1. 概述
本系统旨在为公司员工提供便捷的车辆预约服务，支持在线查看可用车辆、预约使用、管理预约记录等功能。

## 2. 目标用户
- 员工：可预约车辆。
- 管理员：管理车辆、审批预约、查看使用记录。

## 3. 核心功能
- 车辆管理：添加、编辑、删除车辆信息（车牌号、车型、座位数、状态等）。
- 预约管理：员工选择车辆、时间段提交预约，管理员可审批或拒绝。
- 日历视图：查看车辆可用性。
- 通知：预约成功、审批结果通知。

## 4. 数据实体（初步）
- 车辆：id, 车牌号, 车型, 座位数, 状态（可用/维修/已预约）, 创建时间
- 用户：id, 姓名, 部门, 角色（员工/管理员）, 联系方式
- 预约：id, 用户id, 车辆id, 开始时间, 结束时间, 目的, 状态（待审批/已批准/已拒绝/已取消）, 审批人id, 审批备注, 创建时间

## 5. 业务规则（初步）
- 同一车辆在同一时间段内只能被一个预约占用。
- 预约需管理员审批。
- 员工可取消自己的待审批预约。

## 6. 非功能需求
- 响应时间：页面加载 < 2秒。
- 支持并发：预计同时在线用户不超过 100。

## 7. 系统边界
- 无外部系统集成。
- 部署在公司内网。
